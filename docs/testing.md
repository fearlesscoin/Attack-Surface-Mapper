# Testing

## Testing Environment
The project was tested in controlled and authorized environments, including:
- Local Apache web server
- DVWA (Damn Vulnerable Web Application)
- Passive testing on public websites

## Testing Approach
- Verified correct endpoint discovery
- Confirmed network exposure detection
- Validated risk classification logic
- Ensured the tool performs no exploitation

## Ethical Considerations
No intrusive actions such as brute-force attacks or exploitation were performed.
Passive mode was used for public websites to avoid aggressive scanning.

## Post-Testing Cleanup
After testing, all services and ports opened for testing purposes were stopped
and disabled to prevent unintended security exposure.
